Charcoal is the rendering module of the Cartograph library.

DEPENDENCIES:
Epsilon (must be provided)
glfw (must be provided)
stb_image (contained)
free_type (contained)
glew (must be provided)

CONTENT:
core
	Basic Resource manager
glcore
	Blendstates
	Samplerstates
	Rasterstates
	Textures + Management
	Device
	Shaders + Management
render
	instancerenderer (Sprites and models, emulates OpenGl immediate mode)
	

	
CARTOGRAPH
grids
	2D uniform
	2D uniform hexagonal
